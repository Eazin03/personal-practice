各类衣物清洗教程
说明：本教程适配各类常见衣物材质，重点标注清洗禁忌、水温、洗涤剂选择及晾晒要点，简洁易懂，可直接参考操作。
一、棉麻类衣物（衬衫、T恤、裤子、床品等）

1. 清洗前：检查衣物口袋，取出杂物；领口、袖口等易脏部位，可提前用衣领净或洗衣液涂抹，静置5-10分钟预处理。

2. 洗涤方式：可机洗（轻柔/标准模式）或手洗；水温控制在30-40℃，避免高温热水（会导致缩水、褪色）。

3. 洗涤剂：选用中性或弱碱性洗衣液，避免使用含漂白剂的洗涤剂（会损伤面料、导致发黄）。

4. 注意事项：不可长时间浸泡（建议不超过30分钟）；机洗时避免与尖锐物品同洗，防止勾丝；棉麻易皱，洗后可轻轻拉平再晾晒。

5. 晾晒：阴凉通风处平铺或悬挂晾晒，避免暴晒（暴晒会导致褪色、面料变脆）；尽量反面晾晒，保护衣物正面色泽。

二、真丝类衣物（衬衫、连衣裙、围巾等）

1. 清洗前：必须手洗，严禁机洗（机洗会导致面料起球、破损、变形）；不可用硬毛刷擦拭，避免刮伤面料。

2. 洗涤方式：水温控制在20-30℃（冷水或温水），将中性真丝专用洗涤剂稀释后，放入衣物轻轻按压清洗，不可揉搓、拧绞。

3. 洗涤剂：专用真丝洗涤剂最佳，也可选用中性洗衣液，严禁使用碱性洗涤剂（肥皂、洗衣粉）、漂白剂、柔顺剂。

4. 注意事项：浸泡时间不超过10分钟；清洗后用清水轻轻漂洗2-3次，直至无泡沫；不可用力拧干，可用干净毛巾包裹衣物，轻轻按压吸干水分。

5. 晾晒：阴凉通风处悬挂晾晒（用宽肩衣架，避免衣架痕迹），严禁暴晒；反面晾晒，避免面料褪色、变硬。

三、羊毛/羊绒类衣物（毛衣、大衣、围巾等）

1. 清洗前：优先干洗（羊绒大衣、贵重羊毛制品建议送专业干洗店）；可水洗的羊毛衫，需提前查看衣物水洗标，确认可水洗再操作。

2. 洗涤方式：手洗为主，水温20-30℃，用羊毛专用中性洗涤剂，轻轻按压清洗，不可揉搓、拧绞（会导致起球、缩水、变形）。

3. 洗涤剂：羊毛专用洗涤剂，严禁使用碱性、漂白类洗涤剂，避免损伤毛纤维。

4. 注意事项：浸泡时间不超过15分钟；清洗后用清水漂洗干净，可加入少量羊毛柔顺剂，提升面料柔软度；拧干时用毛巾包裹按压，不可用力拧绞。

5. 晾晒：阴凉通风处平铺晾晒（避免悬挂，防止拉伸变形）；晾晒至半干时，可轻轻整理衣物形状，自然晾干；严禁暴晒、烘干。

6. 后续护理：晾干后轻轻拍打，去除褶皱；起球可使用毛球修剪器轻轻修剪，避免用力拉扯。

四、化纤类衣物（聚酯纤维、锦纶、氨纶等，如运动服、外套、打底衫）

1. 清洗前：检查水洗标，大部分可机洗；易脏部位可提前预处理。

2. 洗涤方式：机洗（轻柔/标准模式）或手洗，水温30-40℃，避免高温（高温会导致面料变形、老化）。

3. 洗涤剂：中性或弱碱性洗衣液均可，避免使用漂白剂（部分化纤面料会褪色）。

4. 注意事项：机洗时与其他衣物分开，避免染色；氨纶面料不可用力拧绞，防止失去弹性；避免长时间浸泡（不超过30分钟）。

5. 晾晒：阴凉通风处悬挂晾晒，可暴晒（部分化纤面料耐晒），但深色衣物建议反面晾晒，防止褪色。

五、羽绒服（短款、长款羽绒服）

1. 清洗前：检查衣物水洗标，确认可机洗/手洗；提前拍打羽绒服，使羽绒蓬松；取出内胆（如有），分开清洗。

2. 洗涤方式：可机洗（羽绒服专用模式/轻柔模式）或手洗；水温30℃左右，用羽绒服专用洗涤剂（中性），不可使用碱性洗涤剂、漂白剂。

3. 注意事项：机洗时，可放入洗衣袋，避免羽绒服变形、羽绒结块；清洗时轻轻按压，不可揉搓、拧绞；浸泡时间不超过20分钟。

4. 漂洗与脱水：清水漂洗2-3次，直至无泡沫；脱水时用低速脱水（机洗），或用毛巾包裹按压吸干水分（手洗），不可高速脱水、用力拧绞。

5. 晾晒：阴凉通风处悬挂晾晒，避免暴晒、烘干；晾晒过程中，每隔1-2小时轻轻拍打羽绒服，使羽绒均匀分布，防止结块，直至完全晾干。

六、牛仔类衣物（牛仔裤、牛仔外套）

1. 清洗前：首次清洗可在冷水中加入少量盐或白醋，浸泡10分钟，防止褪色；反面清洗，减少正面磨损、褪色。

2. 洗涤方式：机洗（轻柔模式）或手洗，水温30℃左右，避免高温（高温会导致缩水、褪色严重）。

3. 洗涤剂：中性洗衣液，避免使用漂白剂、强碱性洗涤剂（会导致面料变硬、褪色）。

4. 注意事项：不可长时间浸泡（不超过20分钟）；机洗时避免与浅色衣物同洗，防止染色；清洗后不可用力拧绞，轻轻按压吸干水分。

5. 晾晒：阴凉通风处反面悬挂晾晒，避免暴晒（暴晒会导致褪色、面料变硬）；牛仔裤可自然晾干，无需熨烫，保持版型。

七、通用注意事项

1. 所有衣物清洗前，务必查看水洗标，按照水洗标指示操作（重点关注能否机洗、水温、洗涤剂禁忌）。

2. 深色衣物与浅色衣物分开清洗，避免染色；易掉色衣物首次清洗单独洗。

3. 衣物上的污渍（油渍、污渍、血渍等），需及时处理，避免干涸后难以清洗；血渍用冷水冲洗，不可用热水（会使血渍凝固）。

4. 熨烫：棉麻、真丝、羊毛类衣物可熨烫，化纤、羽绒服不可熨烫；熨烫时控制好温度，真丝、羊毛用低温熨烫，棉麻可用中高温熨烫。

5. 存放：清洗晾干后，折叠整齐或悬挂存放；羊毛、真丝类衣物需防虫蛀，可放入樟脑丸（远离衣物，避免直接接触）。
